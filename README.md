# 🚀 Java Problem Solving Journey

Welcome to my Java Programming repository! This project is a comprehensive collection of problems and solutions designed to master the fundamentals of Java, from basic syntax to Object-Oriented Programming (OOP) concepts.



## 📌 Project Overview
This repository serves as a personal log of my progress in Java. It covers a wide range of logical challenges, mathematical computations, and real-world simulations like Tax Calculators, Logic Gate Simulators and many more wonderful problems with explanatory comments in it.

## 🛠️ Core Concepts Covered
- **Basics:** Variables, Data Types (Primitive & Wrapper Classes).
- **Operators:** Arithmetic, Logical, Ternary, and Bitwise.
- **Control Flow:** `if-else` ladders, Modern `switch` expressions (Java 14+), and Loops (`do-while`, `for`).
- **Data Structures:** Array manipulation and searching algorithms.
- **Methods:** Reusability, Parameters vs. Arguments, and Overloading.
- **OOP Foundations:** Access Modifiers, Encapsulation, and Classes.
