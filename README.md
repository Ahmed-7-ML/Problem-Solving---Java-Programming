## 📌 Project Overview
This repository is my personal **Log of Progress**. It’s not just code; it’s a journey through logical challenges, mathematical computations, and real-world simulations. 

Each program is crafted with **detailed explanatory comments** to make the logic crystal clear for anyone reading the code.

---

## 🛠️ Core Concepts Covered

| Category | Topics |
| :--- | :--- |
| **🏗️ Basics** | Variables, Data Types (Primitive & Wrapper Classes) |
| **⚡ Operators** | Arithmetic, Logical, Ternary, and Bitwise |
| **🔄 Control Flow** | `if-else` ladders, Modern `switch` (Java 14+), `do-while`, `for` loops |
| **📊 Data Structures** | Array manipulation, Searching algorithms, Frequency counters |
| **⚙️ Methods** | Reusability, Parameters vs. Arguments, Method Overloading |
| **🛡️ OOP Foundations** | Access Modifiers, Encapsulation, Classes, and Objects |

## 📂 Highlighted Simulations
> Some of the most interesting problems solved here:

* **Tax Calculator:** Real-world progressive tax slab computation with cess.
* **Logical Gate Simulator:** Simulating AND, OR, and NOT gates using bitwise logic.
* **Leap Year Checker:** Advanced nested logic for calendar computations.
* **Number to Words:** Leveraging modern Java Switch Expressions.
